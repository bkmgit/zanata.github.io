About

Zanata is a web-based translation platform for managing localization projects. Localization (L10n) is the process of translating software or documentation into different languages.

Zanata is written in Java and uses modern web technologies such as JBoss EAP, Seam, GWT, ReactJS and Hibernate. It also presents a REST API (link). It currently supports software and documentation translation of various file types. See here for supported file types in Zanata. Projects can be uploaded to and downloaded from a Zanata server using a Maven plugin or a command-line client.

A public instance of Zanata is hosted on https://translate.zanata.org which accepts content from open source projects for translation.

For projects hosted on jboss.org, use the JBoss instance of Zanata at translate.jboss.org.

Zanata is free software licensed under the LGPL.

For more information see the Documentation available at docs.zanata.org

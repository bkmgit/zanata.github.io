About

Zanata is a web-based translation platform for managing localization projects. Localization (L10n) is the process of translating software or documentation into different languages.

Zanata is written in Java and JavaScript, and uses the following modern web technologies:

Zanata presents a REST API and currently supports software and documentation translation of various file types.

You can upload to and download from Zanata using a Maven plugin, command-line client, or through the website.

A public instance of Zanata is hosted on https://translate.zanata.org which accepts content from open source projects for translation.

For projects hosted on jboss.org, use the JBoss instance of Zanata at translate.jboss.org. For more projects, see the Instances page.

JBoss Decommissioned

The Zanata instance for JBoss.org (translate.jboss.org) has been decommissioned.

New JBoss projects should use one of these instances:

  • translate.zanata.org - Open source projects
  • vendors .zanata.redhat.com - JBoss projects
